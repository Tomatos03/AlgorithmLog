# [812. 最大三角形面积](https://leetcode.cn/problems/largest-triangle-area/)

## 解题思路

1. 输入数据$n$比较小, 考虑直接枚举所有的可能
1. 已知三点计算三角形面积公式$S = \frac{1}{2} |x_1(y_2-  y_1) + x_2(y_3 - y_1) + x_3(y_1 - y_2)|$


## 参考代码

+ 时间复杂度$O(n^3)$


```java
import static java.lang.Math.*;

class Solution {
    public double largestTriangleArea(int[][] points) {
        int n = points.length;
        double sum = 0;
        System.out.println(calSum(points[0], points[1], points[2]));
        for (int i = 0; i < n; ++i) {
            for (int j = i + 1; j < n; ++j) {
                for (int k = j + 1; k < n; ++k) {
                    sum = max(sum, calSum(points[i], points[j], points[k]));
                }
            }
        }
        return sum;
    }

    public double calSum(int []a, int []b, int []c) {
        return 0.5 * abs(a[0] * (b[1] - c[1]) + b[0] * (c[1] - a[1]) + c[0] * (a[1] - b[1]));
    }
}
```
