# [966. 元音拼写检查器](https://leetcode.cn/problems/vowel-spellchecker/)

## 解题思路

1. 模拟思路：使用一个 `set` 存储 `wordList` 中所有原始字符串，使用两个 `map` 分别存储：
   1. `wordList` 中所有字符串转换为小写后的字符串索引映射关系；
   2. `wordList` 中所有字符串转换为小写并将元音字母替换为通配符后的索引映射关系


## 参考代码

+ 时间复杂度$O(nlen(s))$, $len(s)$表示输入字符串的长度


```cpp
using i32 = int;
using i64 = long long;
using pii = pair<int, int>;
using pll = pair<i64, i64>;
using tdii = tuple<double, int, int>;
const int mod = 1000000007, mod0 = 998244353;
class Solution {
    void lower_str(string &str) {
        for (char &c : str) {
            c = tolower(c);
        }
    }
public:
    vector<string> spellchecker(vector<string>& wordlist, vector<string>& queries) {
        string t = "aeiou";

        set<string> st0;
        map<string, int> mp0, mp1;
        int n = wordlist.size();
        for (int i = 0; i < n; ++i) {
            st0.insert({wordlist[i]});

            string lower_str = "";
            string str = "";
            for (char c : wordlist[i]) {
                char c0 = tolower(c);
                lower_str.push_back(c0);

                char c1 = t.find(c0) == string::npos ? c0 : '_';
                str.push_back(c1);
            }

            if (!mp0.count(lower_str)) {
                mp0[lower_str] = i;
            }
            if (!mp1.count(str)) {
                mp1[str] = i;
            }
        }

        vector<string> ans;
        for (string str : queries) {
            if (st0.count(str)) {
                ans.push_back(str);
                continue;
            }

            lower_str(str);
            if (mp0.count(str)) {
                ans.push_back(wordlist[mp0[str]]);
                continue;
            }

            for (char &c : str) {
                if (t.find(c) != string::npos) {
                    c = '_';
                }
            }

            if (mp1.count(str)) {
                ans.push_back(wordlist[mp1[str]]);
            } else {
                ans.push_back("");
            }
        }
        return ans;
    }
};

```
