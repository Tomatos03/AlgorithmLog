# [1518. 换水问题](https://leetcode.cn/problems/water-bottles/)

## 解题思路

1. $numBottles$个空瓶子能够换$\lfloor \frac{numBottles}{numExchange} \rfloor$个饮料, 换了饮料再喝完, 总共拥有的空瓶子数量为$numBottles \bmod numExchange + \lfloor \frac{numBottles}{numExchange} \rfloor$


## 参考代码

+ 时间复杂度$O(\log{numBottles})$


```java
class Solution {
    public int numWaterBottles(int numBottles, int numExchange) {
        int ans = numBottles;
        while (numBottles >= numExchange) {
            int v = numBottles / numExchange;
            int r = numBottles % numExchange;
            ans += v;
            numBottles = v + r;
        }
        return ans;
    }
}
```
