# [200. 岛屿数量](https://leetcode.cn/problems/number-of-islands/)

## 解题思路

1. 输入矩阵能够看作网格图, 题目所求转换为在网格图中数连通块的个数, 可以用并查集求解这个问题


## 参考代码

+ 时间复杂度$O(n^3)$


```cpp
using i32 = int;
using i64 = long long;
using pii = pair<int, int>;
using pll = pair<i64, i64>;
using tdii = tuple<double, int, int>;
const int mod = 1000000007, mod0 = 998244353;
class Solution {
    int m;
    int cal(int x, int y) {
        return x * m + y;
    }

    int find(vector<int>& b, int x) {
        return b[x] == x ? x : find(b, b[x]);
    }

public:
    int numIslands(vector<vector<char>>& grid) {
        vector<vector<char>> &g = grid;
        int n = g.size();
        this->m = g[0].size();

        vector<int> b(n * m);
        iota(b.begin(), b.end(), 0);

        int dx[] = {0, 0, -1, 1}, dy[] = {1, -1, 0, 0};
        int ans = n * m;
        for (int i = 0; i < n; ++i) {
            for (int j = 0; j < m; ++j) {
                if (g[i][j] == '0') {
                    --ans;
                    continue;
                }

                int pos0 = cal(i, j);
                for (int k = 0; k < 4; ++k) {
                    int x = i + dx[k], y = j + dy[k];

                    if (x < 0 || y < 0 || x >= n || y >= m || g[x][y] == '0')
                        continue;

                    int pos1 = cal(x, y);
                    int p0 = find(b, pos0), p1 = find(b, pos1);
                    if (p0 == p1)
                        continue;

                    b[p0] = p1;
                    --ans;
                }
            }
        }

        return ans;
    }
};

```
