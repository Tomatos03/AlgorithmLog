# [231. 2 的幂](https://leetcode.cn/problems/power-of-two/)

## 解题思路

1. $2^x = n$, 其中必然有$x \geq 0(n是整数)$, $n > 0$
1. 如果$n$是$2$的幂, 那么只有一个二进制位上是$1$, 此时必然有$n \& (n - 1) = 0$

```cpp
using i32 = int;
using i64 = long long;
class Solution {
public:
    bool isPowerOfTwo(int n) {
        if (n <= 0)
            return false;

        return (n & (n - 1)) == 0;
    }
};
```
